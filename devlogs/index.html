Shellpkg Devlog - 01/22

By Jocadbz




New vital function and a new release


Hello everyone, i have some cool news to share. Today marks an new sub-release
for shellpkg.

The local package installation function has been added. This is an thing i "imported" from
the legacy shellpkg, which means that if you build an personal package, you can install it
without having to upload on the repos. Be aware of the syntax, that is kinda different to avoid complications
betwwen functions.
That's it for today, enjoy.
