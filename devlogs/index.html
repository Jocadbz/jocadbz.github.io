Hello everyone, i have some cool news to share. After an little time off

New function, and an new release

I had this idea an while back then. It consists of adding an txt file to the repos,
so everytime the "info" function is called, the system downloads the txt file, prints it
on the screen, and deletes the file. Pretty simple, but acctually darn cool. And plus, i'm giving
the mantainers more work to do. ;)
So, i'm gonna finish up wrapping up things and i expect to release the new "1.1.0" version in an few hours
after this post goes up on the site.


PKGfest has started

You see, my package manager is cool but it only has two packages... That is why i'm launching this "event"
so anyone interested can reach out the wiki and create packages. The people who created all the packages during the
PKGfest will have their name forever listed on the README.md file of the repo. So if you are interested, just check
on the github page of shellpkg.
