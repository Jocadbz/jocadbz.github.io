You should stop using VS Code and use Atom

I think everyone I know uses Visual Studio Code to program. And he's not bad.
But he has a lot of problems that I can't ignore, so I've been looking for an
alternative. And I think I finally found it.

VS Code issues

It works very well, I have no doubts. But it consumes a lot of PC resources,
which makes it very heavy on the machine.The support for GIT and Github isn't the
best either, and I was often confused if I had already pushed the repository or not,
and resorted to GIT in the terminal to find out. It also has Microsoft Spyware in it,
which can only be removed if you compile straight from the sourceor download VS Codium,
which is essentially the VS Code package compiled without spyware.


What is the Atom better at?

In fact, Atom was the main code editor on Github, at the time it had not yet
been purchased by microsoft. So they focused a lot on collaborative editing and
Github integration. After Microsoft bought Github, she was already developing VS Code,
so Atom development was left to the community. In terms of resources, it consumes
MUCH less resources than VS Code, even though they are both in Electron. It's integration
with git and Github is very well done, offering a window to manage your changes,
to create commits, and push the repository. It also has a built-in window to support
if your SSH key is encrypted (like mine).


In terms of language support?

Well, I usually write in Python and HTML, so I can confirm that these two languages
work well. All I had to do was go to the packages tab of the editor and install
the "ide-python" package and boom, total Python support. Apparently it also supports
other languages. I tested it with Rust, for example.



Atom is much prettier than VS Code too. Just talking.
