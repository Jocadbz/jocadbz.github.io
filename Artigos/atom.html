Você deveria parar de usar VS Code e usar Atom

Acho que todo mundo que eu conheço usa o Visual Studio Code pra programar. E ele não é ruim.
Mas ele tem varíos problemas que eu não consigo ignorar, então venho procurando uma alternativa. E eu acho que eu finalmente achei.

Os problemas do VS Code

Ele funciona muito bem, não tenho Duvídas. Mas ele consome muitos recursos do PC, o que faz ele ficar bem pesado na maquína.
O suporte a GIT e github também não é dos melhores, e muitas vezes eu ficava confuso se eu já tinha dado PUSH no repositorio ou não,
e recorria ao GIT no terminal pra saber. Ele também tem Spyware da Microsoft nele, que só pode ser removido se você compilar direto da fonte
ou baixar o VS Codium, ue é essencialmente o pacote do VS Code compilado sem o spyware.


No que o Atom é melhor?

Na verdade, o Atom era o editor de codígo principal do Github, na epóca que ele ainda não tinha sido comprado
pela microsoft. Então, eles focaram bastante na edição colaborativa e na integração com o Github. Depois que a Microsoft
comprou o Github, ela já estava desenvolvendo o VS Code, então o desenvolvimento do Atom ficou com a comunidade.
Em termo de recursos, ele consome BEM menos recursos do que o VS Code, mesmo que eles dois sejam em Electron.
Sua integração com o git e o Github é muito bem feita, oferendo uma janela para gerenciar suas mudanças, para criar commits,
e dar push no reposítorio. Também tem uma janela integrada para dar suporte se sua chave SSH é criptografada (como a minha).


Em termos de suportes a linguagens?

Bom, eu normalmente escrevo em Python e HTML, então posso confirmar que essas duas linguagens funcionam bem.
tudo o que tive que fazer foi ir até a aba pacotes do editor e instalar o pacote "ide-python" e boom, total
suporte a Python. Pelo visto também suporta outras linguagens. Eu testei com Rust, por exemplo.



O Atom é muito mais bonito que o VS Code também. Só falando.
