What do I use to create these stupid websites

This is like one of those videos where people show the PCs that cost 10,000 to
run free fire, so I'll show you my laptop and what I use to create the scripts and this website.

The machine

It's literally a regular laptop: Intel Core i3-6006U, 4GB RAM, 1TB HD, running
LMDE 4 with i3wm. You are right, he's relatively weak, but he's good enough for me,
especially since the programs I use do not require "power" and the i3 is very light.
With Thunderbird, Discord, Schildchat, Discord, Vim and some tabs in Firefox, RAM
consumption does not exceed 2GB.


Software

I use things that are very, very conventional. I use Vim as a text editor, Firefox
as my main browser. Some extra apps include Discord and Schildchat (Matrix Client)
as my chat apps. And also the glorious Thunderbird as my email client.


That's pretty much it. I'll update annually so consider this as 2021 edition.
