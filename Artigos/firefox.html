This is probably the biggest article I've ever written.

I will never leave you Firefox

"Oh, but Firefox is bad"
"Oh, but you should use another browser"
Honestly, Firefox remains the best browser. Fast, easy, secure and open-source.
Recently, Firefox has been losing a lot of users, which made me re-evaluate my choice
So, I've tested every browser I can, and this is a solid review of the ones I've tested.

Chromium

I started with the most famous browser there is. Glorious, the benchmark smasher, the amazing... normal browser.
Honestly, Chromium is really cool, more for being just a "ghost" compared to its big brother, Chrome,
it costs some features like synchronization. Not having that kind of resource, I honestly gave up on the spot,
but if you don't care about synchronization, chromium is a great browser.

Gnome Web

So, Gnome Web is not available in my distribution's repositories, so I had to compile it directly from git.
And honestly, it's unusable. It barely works on Youtube, and only performs well on pure HTML sites (like mine).
It is quite broken on some sites, but I see a certain evolution in it little by little. Apart from that they support Firefox Sync,
what a bonus for me. It's definitely cool.

Vivaldi

This was probably one of my best experiences behind Chromium. It is not 100% open-source,
but the fact that some of the parts were open made me kind of...curious. I already used it at the time of Winblows, so I have
an account there and all. AND MY GOD, it's like the KDE of browsers. Everything about it is extremely customizable.
Only he seemed a little slow to me... for some reason. I had the feeling that the pages were loading slower,
so I kind of lost the animation. It also doesn't support detecting whether the OS is using dark mode or not, which is kind of annoying.
It was not the best experience, but compared to other browsers it wins the 3rd place. Definitely take a look at it to
see if you like it or not.

Brave

I didn't like Brave. It puts a lot of things in your face, like the personalized ads you make money watching,
and he also seemed slower to me. There are things to improve, but I don't think I could use it as my main browser.

The veredict:

All the browsers I've tried have a certain glitch that prevents me from using it, except Firefox.
I don't know if it's because I've gotten used to how Firefox works, or because I've been using it since Windows,
but now I can't get out of it.


Firefox remains the best browser for me. Let's see if that changes someday.
