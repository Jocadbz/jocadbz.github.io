Como sair do Vim - Guia atualizado 2022

Eu tenho problemas serios para sair do meu editor de texto preferido, o Vim. Como lembrete para mim mesmo, estou escrevendo esse artigo.

Método 1 - Hard Exit

Primeiramente, com esse metodo, você perde todos os seus dados. É um pequeno preço a se pagar.

Passo 1 - Retire o HD (ou SSD) do seu computador
Passo 2 - Destrua o seu HD (ou SSD) e compre um novo na internet.


Método 2 - Rewrite

Para esse jeito, você precisa ser um chefe de estado ou presidente, caso não seja, use os métodos 1 e 3.

Passo 1 - Descubra a localização de Bram Moolenaar, autor do Vim.
Passo 2 - Extraia informações das senhad do Github dele
Passo 3 - Hackeie os servidores centrais do Github, e adicione um botão de "Sair" no código original do Vim
Passo 4 - Recompile manualmente o Vim do Github e saia do Vim com o botão


Método 3 - Total Nuke

Para esse método, você deve ter acesso a Dinamite e fogo.

Passo 1 - Consiga 10 Kilos de pólvora e jogue em cima do computador
Passo 2 - Fique a uma distancia de cerca de 10 metros
Passo 3 - Exploda o computador
Passo 4 - Compre um novo na internet



Espero que esse guia seja util pra você. Foram os métodos mais fáceis que eu encontrei.
